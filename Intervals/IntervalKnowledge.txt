TECHNIQUES:
1. Intervals is a pair of start time and end time.
2. To work with any interval problem, we need either sort the interval with start time and end time.
3. If we are asked to find overlapping intervals (current start time <= previous end time) or merge overlapping intervals,
   the approach is to sort the intervals by start time.
4. If we are asked to find non-overlapping intervals, the approach is to sort the intervals by end time. Because if you sort by start time, the overlapping
   interval might come first and it will mask all the non-overlapping intervals inside it.
